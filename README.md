# AIアシスタントのフロントエンド

## 事前準備
1. <code>.env.development</code>を作る
```editorconfig
NEXT_PUBLIC_SERVER_HOST=[あなたの バックエンド・サーバー の　URL]
NEXT_PUBLIC_SERVER_HOST_WITH_SSL=FALSE
NEXT_PUBLIC_SUPABASE_URL=[あなたの スーバーベース の　URL]
NEXT_PUBLIC_SUPABASE_ANON_KEY=[あなたの スーバーベース の API KEY]
```

2. Node.js パッケージをインストールする
```bash
npm install
```


## 起動する方法
```bash
./run.sh
```
